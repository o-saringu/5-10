プログラム復習コンテンツ

keyPressed変数

サンプルコードの結果

サンプルコード

　　 void setup(){

　　 　size(240, 120);　　 　

　　 }

　　

　　 void draw(){

　　 　background(204);

　　 　line(20, 20, 220, 100);

　　 　if(keyPressed){

　　 　　line(220, 20, 20, 100);

　　 　}

　　 }

説明文

　keyPressed変数はキーボードのボタンが押されたかを判別する変数である。

　キーボードのボタンが押されると真(true)、押されていないと偽(false)となる。

　キーボードのボタンはどのボタンでもよい。

　この変数は真と偽の値しか算出しないため演算子を用いる必要はない。

　例では、キーボードのボタンを押すと線が追加される仕組みである。
